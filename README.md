# CodeLink 💬

A real-time chat app built by **Rajvi Patel** using **Node.js**, **Express**, and **Socket.io**.

## 🚀 Features
- Join chat rooms instantly
- Real-time messages using WebSockets
- Custom bot announcements (CodeLink Bot)
- Simple and clean interface

## 🛠️ Tech Stack
- Node.js
- Express.js
- Socket.io
- HTML, CSS, and Vanilla JavaScript (frontend)

## 💻 Run Locally
```bash
git clone https://github.com/Rajvi33/codelink.git
cd codelink
npm install
npm start
